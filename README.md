## Documentation

**Installation, configuration, features, and troubleshooting** live under **[docs/](docs/)** — start at **[docs/index.md](docs/index.md)** (links to setup, integration, and troubleshooting pages) so content stays valid when the tree is mirrored or published alone.

Quick install:

```bash
cd frappe-bench
bench get-app https://github.com/washmoredevelopment/taxjar_erpnext.git
bench --site your-site.localhost install-app taxjar_erpnext
bench --site your-site.localhost migrate
```

For ERPNext setup, architecture notes, and license/acknowledgements, follow the links from [docs/index.md](docs/index.md).
